Pool Table Management Application



For our first project we were tasked with creating a management application using the python language. in the Digital Crafts curriculum we were giving the task to create a game room management application for the employees of the cougar game room. This application would need to keep track and save the of the following information:

  • Pool table number
  • Rent start time
  • Rent end time
  • Total time played
  • Cost of time played
